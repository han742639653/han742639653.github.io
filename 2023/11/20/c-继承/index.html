c++ 继承

在C++中，继承是一种面向对象编程的重要概念，它允许一个类（称为派生类）从另一个类（称为基类）那里继承属性（数据）和行为（方法）。这使得代码重用变得更加容易，并且有助于建立层次化的类关系。

上述代码中Rectangle 类通过 public 关键字继承了 Shape 类。这意味着 Rectangle 类可以访问 Shape 类中的 setWidth 和 setHeight 方法，可以通过继承的数据成员 width 和 height 计算矩形的面积。

继承还可以分为不同的类型，比如公有继承、私有继承和保护继承，它们决定了派生类对基类成员的访问权限。此外，C++ 还提供了虚继承和多重继承等高级特性，以满足更复杂的类关系需求。

多重继承

多重继承指的是一个派生类同时继承自多个基类。这种特性使得派生类可以获得多个基类的属性和行为，从而支持更加丰富的类关系
